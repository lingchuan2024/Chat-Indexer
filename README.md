# AI Chat 目录导航

为 ChatGPT、DeepSeek、Kimi、Gemini 对话页面自动生成可折叠目录，支持搜索、快速跳转和阅读位置追踪。

## 支持的平台

| 平台     | 地址                                      |
| -------- | ----------------------------------------- |
| ChatGPT  | chatgpt.com / chat.openai.com             |
| DeepSeek | chat.deepseek.com                         |
| Kimi     | kimi.ai / kimi.moonshot.cn                |
| Gemini   | gemini.google.com                         |

当前版本使用“平台配置 + 通用 DOM 启发式”的混合适配方式。
ChatGPT 使用专用属性适配，DeepSeek、Kimi、Gemini 使用平台配置加通用消息扫描回退。

## 安装

### Edge / Chrome（Chromium 内核）

1. 打开浏览器，地址栏输入 `edge://extensions/` 或 `chrome://extensions/`
2. 右上角开启 **「开发人员模式」**
3. 点击 **「加载解压缩的扩展」**
4. 选择本插件目录（包含 `manifest.json` 的文件夹）
5. 打开任一支持的 AI 对话页面，右侧即出现目录面板

## 使用

### 目录面板

- 每个**用户提问**自动成为一个目录项
- 如果 AI 回复中包含 Markdown 标题（`##`、`###` 等），点击问题左侧的 `▶` 可展开查看
- 点击问题文字跳转到对话中对应位置，目标会有绿色高亮
- 点击 `▶` 箭头仅展开/折叠子标题，不跳转

### 阅读位置追踪

滚动对话时，目录中当前正在阅读的问题会自动高亮（紫色左边框 + 深色背景），目录列表也会自动跟随滚动。

### 搜索

顶部搜索框输入关键词，实时过滤目录。搜索范围覆盖用户提问、AI 回复中的标题、以及 AI 回复的正文内容。

### 面板操作

| 操作      | 方式                                    |
| --------- | --------------------------------------- |
| 折叠/展开 | 点击右上角 `>` 按钮                   |
| 调宽度    | 鼠标移到面板左边缘拖拽（160px ~ 420px） |
| 拖位置    | 按住面板顶部标题栏上下拖动              |
| 恢复面板  | 折叠后右下角出现圆形按钮，点击展开      |

面板宽度、位置、折叠状态会自动保存，刷新页面后恢复。

## 项目结构

```
├── manifest.json     # Chrome 扩展配置
├── content.js        # 入口：初始化、SPA 导航
├── platforms.js      # 平台检测与角色识别
├── scanner.js        # DOM 扫描与目录数据提取
├── renderer.js       # 目录渲染与阅读位置高亮
├── layout.js         # 侧边栏 UI、拖拽、缩放、设置持久化
├── utils.js          # 工具函数
├── content.css       # 样式
└── icons/            # 图标
```
